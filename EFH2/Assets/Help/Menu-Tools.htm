Hydrologic Soil Group

This command is available only when the ‘RCN’ tab is activated. Use this command to open the hydrologic soil group lookup dialog window. This window allows the user to view the data in the SOILS.HG file to lookup the hydrologic group value, A, B, C, or D for various soil types.
